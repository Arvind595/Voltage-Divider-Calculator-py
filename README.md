# Voltage-Divider-Calculator-py

## Voltage Divider with 3 Resistors

This setup uses three resistors (R1, R2, R3) to divide an input voltage (`Vin`) into two tap voltages: `Vout1` and `Vout2`.

### Circuit Diagram

```
Vin --- R1 ---+--- R2 ---+--- R3 --- GND  
              |          |
           Vout1      Vout2
```
